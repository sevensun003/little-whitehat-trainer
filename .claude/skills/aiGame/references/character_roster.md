## 四、群演批量生成策略

为 C7 面馆关准备:
- 做一个 `base_robot.svg`,结构只保留"空心困困脸+梯形身体+黑描边"
- 代码里用 CSS `fill` 或 SVG `filter` 动态染色
- 装饰元素(兔耳/尖角/光头等)可从装饰库随机组合
- 50 个群演 = 50 次染色变换,无需实际生成 50 张图

这是极简风格的巨大优势,必须充分利用。

## 五、角色命名规则(新增角色时)

未来如果需要新增角色(如模拟赛的 NPC),遵守:

1. 从用户提供的原始清单中优先挑选未使用的名字
2. 英文名用拼音或意译,保持**短**(方便朗读)
3. 避免与已存在角色视觉冲突(主色或头顶装饰不能重复)
4. 命名格式:中文名(英文名,如"特纳 Tunner")
5. 新增角色必须同步更新本文件 + 新建 prompt 档案
