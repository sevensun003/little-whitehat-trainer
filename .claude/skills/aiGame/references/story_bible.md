## 世界观设定

**指令星球**:所有机器人居住的星球,通过"指令"完成工作。

**中央指令塔**:星球核心建筑,AI 主机**小天**住在塔里,负责给全星球机器人发指令。

**故事开始**:某天中央指令塔被神秘入侵,小天的指令系统出了 bug。机器人们开始走错路、送错件、认错人。星球陷入混乱。

**主角登场**:**婉婉**,7 岁小女孩,戴白色鸭舌帽。被小天紧急召唤——只有"白帽子"能进入指令的底层,把被篡改的指令一条条修复回来。

**主角装备**:一本《指令手册》(五类指令卡)。

**地点**:樱桃镇(樱花、木屋、菜市场、学校)
**主线**:熟悉指令,认识 3 位机器人伙伴。

### T1 · 醒来的婉婉 (教学:方向)

**场景**:婉婉的卧室
**出场角色**:婉婉、小天(通过电视呼叫)

**开场对话**:
```
小天:"婉婉!醒醒!"
小天:"指令星球出大事啦!"
小天:"快到门口来,我告诉你!"
旁白:"用方向指令,让婉婉走到门口。"
```

**通关对话**:
```
小天:"做得好!"
小天:"你已经学会了方向指令。"
小天:"指令星球被坏人搞乱了,需要你帮忙!"
小天:"准备好出发了吗?"
```

### C1 · 被偷改的购物清单 (概念:指令注入)

**场景**:超市入口
**出场角色**:平基加入队伍 ⭐、爸爸(背景)

**开场对话**:
```
平基:"婉婉!有坏事发生!"
平基:"你爸爸的购物清单被改了。"
平基:"他本来只买牛奶。"
平基:"现在清单上却有 100 个冰淇淋!"
小天:"坏人偷偷加了一条指令。"
小天:"帮忙找出多余的那一条,删掉它!"
```

**核心玩法**:屏幕显示一串已有指令队列,玩家要识别并删除"买冰淇淋"那条多余指令,让爸爸只买牛奶

**通关对话**:
```
爸爸(憨厚):"哎呀......差点被骗。"
小天:"这叫'指令注入'。"
小天:"坏人把多余的指令塞进正常指令里。"
小天:"就像有人在你妈妈的字条上偷加字一样。"
平基:"婉婉,我跟你一起去!"
```
**成就解锁**:平基加入队伍 🎉

### C3 · 会传话的邮筒 (概念:中间人攻击)

**场景**:樱桃镇邮筒广场
**出场角色**:奥伦加入 ⭐

**开场对话**:
```
奥伦:"我是奥伦,信号中继员。"
奥伦:"我负责把镇上的消息转发出去。"
奥伦(困惑):"可最近我传的消息都......变样了?"
小天:"有反射镜在中间偷偷改消息!"
小天:"找出来,绕开它。"
```

**核心玩法**:信号从 A 射向 B,中途有一面反射镜改变方向/颜色,玩家要先移除反射镜再让信号通过

**通关对话**:
```
奥伦:"原来是这样!"
小天:"这叫'中间人攻击'。"
小天:"有人悄悄夹在你们中间。"
小天:"会偷听或者改你们的话。"
小天:"就像传纸条时前排同学偷看。"
奥伦:"带我一起去!我能帮你们中转消息。"
```
**成就解锁**:奥伦加入队伍 🎉 三人组集结完毕

### C4 · 假扮的水电工 (概念:社会工程学)

**场景**:迷雾森林入口
**出场角色**:杰文(森林守门员)、克拉克(真水管工,被骗者)

**开场对话**:
```
杰文(懒洋洋):"进森林要登记哦。"
平基:"我们看到有个'水电工'刚进去了..."
克拉克(气喘吁吁跑来):"我才是水电工!"
克拉克:"刚才那个是假的!"
小天:"坏人假扮身份骗过了杰文!"
```

**核心玩法**:让杰文"再确认一次身份"——要求真克拉克走过杰文面前,杰文对比后识别出假的那个(已离开)的破绽

**通关对话**:
```
杰文(清醒了):"居然被骗了..."
小天:"这叫'社会工程学'。"
小天:"坏人不攻击系统,攻击你的'相信'。"
小天:"就像坏人按门铃说'我是叔叔你爸叫的'。"
小天:"记住:不要只凭一句话就相信。"
```

## 对话文案 6-9 岁友好规范(必须遵守)

1. **每句 ≤ 20 字**
2. **主谓宾齐全**,不倒装不省略
3. **每段不超过 4 句**,超过要分段弹出
4. **避免抽象词**:
   - 权限 → 准入证
   - 协议 → 约定
   - 漏洞 → 破洞
   - 凭证 → 钥匙/身份
   - 注入 → 塞进去
   - 攻击 → 搞乱
5. **所有对话必须配朗读按钮**
6. **每个概念关通关对话必须包含**:
   - 概念名称
   - 生活类比
   - 防御建议
